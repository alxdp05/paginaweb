Maquina Expendedora

Este es el manual de una maquina expendedora.

Información

La máquina expendedora cuenta con un sistema de configuración de contraseña para hasta 2 usuarios. Si no se ha establecido una contraseña, el usuario puede configurar una de hasta 8 caracteres, la cual se guarda en el EEPROM de Arduino. Además, se puede ingresar un código de 5 caracteres, y si es válido, se almacena en el EEPROM de Arduino para evitar ingresos repetidos. Los créditos también se guardan en el EEPROM, y es importante destacar que los créditos y las contraseñas de los usuarios se almacenan por separado en el EEPROM.

Cuando hayas metido un codigo valido tiene la funcionalidad de guardar los creditos para mas adelante en el sistema de usuarios, si no tienes un usuario puedes meter un codigo y cuando ingrese un codigo valido puede seleccionar el articulo que quiera sacar de la maquina expendedora.

Funcionamiento

El Funcionamiento de esta maquina expendedora es la siguiente:

El sistema cuenta con una función de inicio de sesión que permite a los usuarios acceder al sistema mediante el uso de hasta dos cuentas de usuario personalizadas. Cada cuenta puede ser protegida con una contraseña única para garantizar la seguridad de los datos.

Además, el sistema ofrece una funcionalidad adicional que permite a los usuarios ingresar códigos especiales para obtener créditos. Estos créditos pueden ser canjeados tanto por usuarios registrados como por aquellos que no tienen una cuenta en el sistema. Es importante destacar que los créditos obtenidos se almacenan de manera separada en cada cuenta de usuario, asegurando que los créditos estén asociados correctamente.
