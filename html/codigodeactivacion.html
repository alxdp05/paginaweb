Generar un código

Este apartado es para generar un código para usarlo en la Máquina Expendedora.

El código utilizado

El código utilizado de este generador es el siguiente:


        function generarCodigo() {
            var codigos = [
                "AAAAA",
                "BBBBB",
                "CCCCC",
                "DDDDD",
                "AAA11",
                "BBB22",
                "CCC11",
                "DDD22",
                "222DD",
                "333DD",
                "444DD"
            ];

            var codigoGenerado = codigos[Math.floor(Math.random() * codigos.length)];

            document.getElementById('codigo-generado').innerHTML = codigoGenerado;
        }
    

        <h1 id="titulo">Generar un código</h1>
        <p id="subtitulo">Este apartado es para generar un código para usarlo en la Máquina Expendedora.</p>
        <button id="button" onclick="generarCodigo()">Generar</button>
        <p id="codigo-generado"></p>
